Usage

After installing all required dependencies, the script can be executed directly. This tool is designed to send a desktop notification with a predefined message. You may modify the content of the notification inside the script according to your needs.

Ensure that all dependencies are installed:

pip install plyer


Run the script from the project directory:

python3 latihan.py


Once executed, the script will display a system notification on your device.
The behavior of the notification may vary depending on your operating system and desktop environment.

To customize the notification text, title, or additional functions, open the script file and adjust the relevant variables or function calls.
