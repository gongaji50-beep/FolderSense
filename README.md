Signal Sender – Python Notification Tool

Simple Python script that can send notifications using the plyer library.
This project is intended for beginners who want to learn how to build and share simple digital utilities.

Features

Sends desktop notifications

Easy-to-read and editable Python code

Lightweight and cross-platform (Linux, Windows, macOS)

Requirements

Before running the script, install the required dependency:

pip install plyer

How to Run

Clone this repository:

git clone https://github.com/username/repository-name.git


Navigate to the project directory:

cd repository-name


Run the script:

python3 latihan.py
